#Hostel App

Hybrid app for small hostel.

##Getting started

Install all packages and build app
```
$ npm i && npm run build
```

Run server
```
$ npm run server
```

Watch changes
```
$ npm run watch
```

##Technologies

* Webpack - https://webpack.github.io/
* Onsen - https://onsen.io/
* React - https://facebook.github.io/react/
